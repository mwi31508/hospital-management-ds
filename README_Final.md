## Project Summary
This project is a console-based Hospital Management System developed in C++.
It demonstrates the practical use of Data Structures in a real-world scenario.

## Data Structures Used
- Queue: Used to manage waiting patients
- Stack: Used to store treated patient records

## Features
- Add patient to waiting list
- Treat patient and move to treated records
- Search patient records
- Update patient information
- Display waiting and treated patients

## How to Run
1. Open the project in any C++ compiler (VS Code, Dev-C++, CodeBlocks).
2. Compile the `main.cpp` file.
3. Run the program.
4. Use menu options to perform operations.
