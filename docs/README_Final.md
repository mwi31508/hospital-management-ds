## Project Summary
This project is a console-based Hospital Management System developed using C++.
It demonstrates the practical use of data structures in managing patient records.

## Data Structures Used
- Queue (for managing waiting patients)
- Stack (for managing treated patients)

## Key Features
- Add patient to waiting list
- Treat patient and move to treated records
- Search patient in waiting or treated list
- Update patient name
- Display waiting and treated patients

## How to Run the Program
1. Open the project in any C++ compiler.
2. Compile the file main.cpp.
3. Run the program.
4. Use menu options to perform operations.
